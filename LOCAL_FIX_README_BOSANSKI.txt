VYLNAX PRO - ISPRAVLJENA LOKALNA VERZIJA
=========================================

1. Obavezno prvo raspakuj cijeli ZIP preko "Extract all".
2. Otvori raspakovani folder.
3. Dvaput klikni START_VYLNAX.bat.
4. Prvi put instalacija paketa moze trajati nekoliko minuta.
5. Ostaviti otvorena dva prozora: VYLNAX BACKEND i VYLNAX FRONTEND.
6. U FRONTEND prozoru skeniraj QR kod kroz Expo Go.
7. Telefon i laptop moraju biti na istom Wi-Fi-ju.
8. Ako Windows Firewall pita, dozvoli Python i Node.js na privatnoj mrezi.

Provjereno:
- Python backend se pokrece i /api/health vraca OK.
- Lokalna prijava radi.
- PZN 06313438 vraca Ibuflam 800 mg Lichtenstein.
- TypeScript provjera prolazi bez gresaka.
- Android Expo bundle se uspjesno izgradi.
